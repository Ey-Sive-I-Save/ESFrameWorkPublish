# ESRuntimeWatch Example

1. Create a GameObject in a scene.
2. Add `Example_RuntimeWatchActor`.
3. Enter Play Mode.
4. Open `ES简单工具 / 【ES集成工具集】 / 运行时观察`.

The page groups watched fields by `战斗/角色`, `战斗/AI`, and `战斗/技能`.
Use the object foldout to inspect values, and click `定位` to select and ping the object that owns the watched fields.
