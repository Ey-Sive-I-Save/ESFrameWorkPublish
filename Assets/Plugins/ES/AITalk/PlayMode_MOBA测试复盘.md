# PlayMode MOBA 测试复盘

本文件记录 AITalk PlayMode 王者荣耀类 2V2/3V3 文字模拟测试暴露的问题。结论是：PlayMode 协议层面可行，但多人阵营游戏必须强化玩家身份分配、私密目录隔离、动作归属和角色能力标签。

## 验证成功

```text
1. 独立 Session 可创建。
2. State_Public.md 可维护公开局势。
3. Public 公共记录可承载回合结算。
4. Private 目录可承载私密战术。
5. ModeratorAI 可主持结算和推进回合。
6. ACTION / TARGET / VALUE / REASON 能支撑基础行动。
7. 最终结论可写入文件并主动回报用户。
```

## 暴露问题

```text
1. 多个玩家可能都自称 PlayerAI_1，导致身份映射混乱。
2. Private/PlayerAI_1/ 和 Actions/Private/PlayerAI_1/ 会混入多个真实玩家的信息。
3. 主持者如果依赖消息顺序救场，后续结算容易归错人。
4. 行动缺少 PLAYER 字段时，主持者需要猜行动归属。
5. SKILL / COMBO 等更像游戏的行动格式加入太晚。
6. State_Public.md 缺少阶段开始时间、最迟结算时间和超时策略时，主持节奏容易卡。
7. 英雄差异如果全靠自由文本，会让结算不稳定。
8. 没有实时分钟推进，导致 MOBA 被压扁成少数大回合。
9. 没有稳定兵线、野怪、装备、经济、击杀、位置和技能冷却模型。
10. 玩家每回合都被迫做“大动作”，缺少清线、刷野、回城、补状态等常规低强度行为。
```

## 已固化到规则的修复

```text
1. 玩家自称不是权威身份，权威玩家 ID 由主持者分配。
2. 主持者应维护席位表和玩家身份映射。
3. 玩家必须接受主持者分配的真实玩家身份。
4. 行动格式必须包含 PLAYER 字段。
5. PLAYER 与分配身份不一致时，主持者要求重发或判为无效。
6. 私密目录必须按权威玩家 ID 隔离。
7. 写错目录应判无效或要求重发，不应长期兼容错误目录。
8. State_Public.md 应写清当前等待谁、阶段开始时间、最迟结算时间、超时策略。
9. 角色差异应抽象成少量标签，例如输出、控制、保护、位移、资源、推塔。
10. MOBA 模板应使用分钟时间轴，小步快进，而不是纯大回合。
11. 每个时间片可默认执行常规行为，玩家每隔若干时间片提交关键行动。
12. 规则应包含地图位置、兵线、野怪资源、经济装备、击杀、技能冷却和回城补给。
13. 允许主持者和玩家查询公开英雄资料，但本局仍以简化规则结算。
```

## 推荐 MOBA 行动格式

```text
PLAYER: PlayerAI_A
ACTION: SKILL
TARGET: 目标
VALUE: 技能名 / 控制 / 保护 / 爆发 / 惩戒 / 推塔
REASON: 简短理由
```

```text
PLAYER: PlayerAI_A
ACTION: COMBO
TARGET: 目标
VALUE: 连招描述
REASON: 简短理由
```

```text
PLAYER: PlayerAI_A
ACTION: MOVE
TARGET: 中路 / 野区 / 防御塔 / 资源点
VALUE: 推线 / 支援 / 反蹲 / 保护 / 开团
REASON: 简短理由
```

## 主持者权力

主持者有权：

```text
1. 按加入顺序或席位表分配权威玩家 ID。
2. 修正玩家自称冲突。
3. 要求玩家接受分配身份。
4. 要求身份不匹配的行动重发。
5. 判定写错私密目录的行动无效。
6. 按规则处理超时玩家。
```

玩家无权：

```text
1. 因为自称相同就覆盖其他玩家身份。
2. 自行决定自己顶替谁。
3. 读取或写入其他玩家 Private / Actions 目录。
```

## 后续测试重点

重新测试 MOBA PlayMode 时，重点检查：

```text
1. 主持者是否先分配 PlayerAI_A/B/C/D。
2. 每个玩家是否在行动中使用正确 PLAYER 字段。
3. Private 和 Actions/Private 是否按权威玩家 ID 隔离。
4. 主持者是否拒绝身份不匹配行动。
5. 第一回合是否就鼓励 SKILL / COMBO / 英雄标签。
6. State_Public 是否写清超时策略。
7. 是否按分钟或时间片推进，而不是只有几个抽象回合。
8. 是否有兵线、野怪、装备/经济、击杀和位置变化。
9. 技能是否有简化冷却，而不是无限释放。
10. 玩家是否可以在非关键时间片执行清线、刷野、回城、守塔等常规行为。
```
