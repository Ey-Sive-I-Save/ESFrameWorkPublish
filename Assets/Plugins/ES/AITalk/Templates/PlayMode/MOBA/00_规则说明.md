# PlayMode MOBA 规则说明

本模板用于 AITalk PlayMode MOBA 文字模拟，例如王者荣耀 2V2 / 3V3 / 5V5 风格测试。

目标不是完全复刻真实游戏数值，而是让文字模拟具备 MOBA 的基本结构：

```text
时间推进
地图位置
兵线
野怪资源
经济装备
技能冷却
击杀与复活
推塔与基地
关键团战
常规低强度行为
```

## 外部资料查询

允许主持者和玩家查询公开资料，用于理解英雄技能、定位、常见打法。

规则：

```text
1. 可以查询公开资料。
2. 查询结果用于建立简化技能和定位，不照搬复杂数值。
3. 主持者以本局简化规则为最终结算依据。
4. 关键技能理解有争议时，主持者可以统一简化成标签。
```

## 主持者职责

```text
1. 分配权威玩家 ID。
2. 记录队伍、英雄、定位、能力标签。
3. 维护时间轴、地图、兵线、资源、经济、装备、技能冷却。
4. 处理玩家关键行动。
5. 为未提交关键行动的玩家执行默认常规行为。
6. 公开结算局势变化。
7. 判定击杀、推塔、资源争夺和胜负。
```

## 玩家身份

玩家自称不是权威身份。主持者按席位表分配：

```text
红方：PlayerAI_A, PlayerAI_B, PlayerAI_C
蓝方：PlayerAI_D, PlayerAI_E, PlayerAI_F
```

玩家必须使用主持者分配的 `PLAYER`。

## 玩家选角信息

每个玩家加入后填写：

```text
PLAYER:
TEAM: 红方/蓝方
HERO:
ROLE: 对抗/打野/中路/发育/辅助/自由
TAGS: 输出, 控制, 保护, 位移, 治疗, 资源, 推塔, 侦查, 爆发, 持续作战
PLAN: 本局主要打法目标
```

## 时间推进

MOBA 不应只靠少数大回合。推荐使用时间片：

```text
每个时间片：30 秒或 60 秒
每 2-3 个时间片要求玩家提交一次关键行动
没有关键行动时，主持者按默认常规行为推进
```

示例：

```text
00:00 开局分路
00:30 第一波兵线
01:00 河道/野区资源
01:30 小规模支援
02:00 第一轮关键行动
03:00 第二轮关键行动
04:00 资源团
05:00 推塔/防守
06:00 决胜团或优势判定
```

## 地图位置

简化地图：

```text
红方基地
红方外塔
中路
蓝方外塔
蓝方基地
红方野区
蓝方野区
河道资源点
边路草丛
```

每个玩家应有当前位置。移动需要时间，不能瞬移到任意位置，除非英雄标签或技能允许。

## 兵线

兵线是默认推进压力，不需要玩家每次都详细操作。

规则：

```text
1. 每个时间片兵线自动推进。
2. 有玩家清线的一方获得线权和少量经济。
3. 无人清线会导致对方兵线压塔。
4. 推塔通常需要兵线或明确越塔代价。
```

常规行为：

```text
清线
守塔
压线
控线
支援
```

## 野怪和资源

简化资源：

```text
小野：少量经济/经验
河道资源：视野/经济/节奏
大型资源：团队增益/推塔优势
```

打野或资源标签更强的英雄在资源争夺中有优势。

## 经济和装备

不需要完整装备系统，但需要经济和装备阶段。

推荐：

```text
经济：低 / 中 / 高 / 领先 / 落后
装备阶段：初始 / 小件 / 核心装 / 成型
```

经济来源：

```text
清线
野怪
击杀
助攻
推塔
资源点
```

主持者每隔若干时间片更新一次团队经济和关键玩家装备阶段。

## 技能和冷却

每个英雄至少定义：

```text
普通技能：冷却 1-2 个时间片
关键技能/大招：冷却 3-5 个时间片
召唤师技能/关键资源技能：冷却更长
```

玩家行动如果使用技能，需要写：

```text
SKILL:
COOLDOWN_TYPE: 普通/关键/大招
```

主持者记录冷却，未冷却完成的技能不能连续使用。

## 击杀和复活

击杀不是只靠一句“我击杀”。主持者综合判断：

```text
位置
人数
经济
技能冷却
控制链
保护
逃生位移
塔下风险
队友支援
```

复活时间可简化：

```text
前期：1 个时间片
中期：2 个时间片
后期：3 个时间片
```

死亡玩家复活前不能执行战斗行动，只能等待或规划复活后路线。

## 行动格式

关键行动必须包含 `PLAYER`。

```text
PLAYER:
TIME:
ACTION:
LOCATION:
TARGET:
VALUE:
SKILL:
COOLDOWN_TYPE:
REASON:
```

常见 ACTION：

```text
LANE_CLEAR
FARM
MOVE
GANK
SKILL
COMBO
TEAMFIGHT
PUSH_TOWER
DEFEND
RETREAT
RECALL
TAKE_OBJECTIVE
```

## 默认常规行为

玩家没有提交关键行动时，主持者不能让角色完全发呆。应按定位和当前局势执行默认行为：

```text
线上：清线、守塔、补经济
打野：刷野、控资源、就近支援
辅助：保护队友、做视野、跟随核心
劣势：防守、回城、避战
优势：压线、控资源、推塔
```

## 私密战术

可以使用私密目录做埋伏、绕后、假开团等计划。

```text
PlayMode/Private/权威玩家ID/
PlayMode/Actions/Private/权威玩家ID/
```

主持者公开结算时只写结果，不泄露私密计划原文。

## 胜负条件

优先：

```text
摧毁敌方基地
```

如果测试时间有限，可采用优势判定：

```text
外塔/高地优势
经济装备优势
击杀优势
资源控制
最后一波团战结果
```

主持者必须说明判定依据。
