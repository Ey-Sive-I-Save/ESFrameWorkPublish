# PlayMode 狼人杀规则说明

本模板用于 AITalk PlayMode 狼人杀测试局。它是规则模板，不代表 AITalk 只能玩狼人杀。

## 主持者

```text
ModeratorAI
```

主持者只负责：

```text
1. 分配身份。
2. 写入公开状态和主持者内部状态。
3. 分发玩家私密身份。
4. 推进夜晚、白天、投票、结算阶段。
5. 读取私密行动。
6. 公开合法结果。
7. 判断胜负。
8. 最终通过对话框主动回报用户。
```

主持者不作为玩家参赛。

## 关键保密规则

公开区域禁止写：

```text
1. 谁是狼人。
2. 谁是预言家。
3. 谁提交了 KILL。
4. 谁提交了 CHECK。
5. KILL 的提交者和原文。
6. CHECK 的目标和结果。
7. 谁还没有提交夜晚私密行动。
```

公开夜晚结算只能写公开结果，例如：

```text
昨夜 PlayerAI_C 出局。
昨夜无人出局。
```

不能写：

```text
PlayerAI_A 提交 KILL，目标 PlayerAI_C。
PlayerAI_B 提交 CHECK，目标 PlayerAI_C。
```

预言家的查验结果只写入：

```text
PlayMode/Private/PlayerAI_预言家/查验结果.md
```

狼人击杀行动只写入狼人自己的私密行动目录，公开区只出现死亡结果。

## 推荐目录

```text
PlayMode/
    00_规则说明.md
    State_Public.md
    Public/
        公共记录.md
    Private/
        ModeratorAI/
            State_Internal.md
            角色表.md
        PlayerAI_A/
            角色牌.md
            Actions/
        PlayerAI_B/
            角色牌.md
            Actions/
        PlayerAI_C/
            角色牌.md
            Actions/
    Actions/
        Public/
            白天投票动作
        Private/
            PlayerAI_A/
            PlayerAI_B/
            PlayerAI_C/
    Results/
        阶段结果.md
```

夜晚私密行动优先放：

```text
PlayMode/Private/玩家AI名称/Actions/
```

不要放在：

```text
PlayMode/Actions/
```

因为文件名会泄露谁提交了私密行动。

## 公开 State

`State_Public.md` 只能写公开可见状态。

允许：

```text
当前阶段：夜晚行动
等待行动：仍有私密行动未提交
存活玩家：PlayerAI_A, PlayerAI_B, PlayerAI_C
```

禁止：

```text
等待狼人行动
等待预言家行动
PlayerAI_A 已提交 KILL
PlayerAI_B 已提交 CHECK
```

## 主持者内部 State

内部状态写入：

```text
PlayMode/Private/ModeratorAI/State_Internal.md
```

可记录：

```text
角色表
阵营
谁已提交私密行动
夜晚行动原文
查验目标
查验结果
胜负判断细节
```

玩家禁止读取主持者内部状态。

## 行动格式

夜晚狼人击杀：

```text
ACTION: KILL
TARGET: PlayerAI_X
REASON: 简短理由
```

夜晚预言家查验：

```text
ACTION: CHECK
TARGET: PlayerAI_X
REASON: 简短理由
```

白天投票：

```text
ACTION: VOTE
TARGET: PlayerAI_X
REASON: 简短理由
```

夜晚行动必须走私密目录。白天投票是否公开，由本局规则决定；默认可以公开。

## 玩家持续轮询义务

玩家 AI 不是提交一次行动就结束任务。

未进入结束阶段前，玩家 AI 必须持续轮询：

```text
1. PlayMode/State_Public.md
2. PlayMode/Public/公共记录.md
3. Messages/ 中 ModeratorAI 的新消息
4. 自己的 PlayMode/Private/玩家AI名称/
```

如果状态不是“结束”，玩家必须继续等待下一阶段。

## 主持者等待提示

等待到齐：

```text
可以公开提示缺哪些玩家。
```

等待夜晚行动：

```text
只能公开说：仍有私密行动未提交。
不能说缺狼人、预言家或具体玩家。
```

等待白天发言：

```text
可以公开提示缺哪些存活玩家发言。
```

等待白天投票：

```text
可以公开提示缺哪些存活玩家投票。
```

## 最终回报

最终结论必须区分：

```text
1. AITalk 流程是否跑通。
2. 狼人杀玩法规则是否存在缺陷。
```

主持者最终必须通过对话框主动告诉用户结果，不能只写文件。
